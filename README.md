**Links/commands:**
To run frontend: **ng serve**
To run backend: **.\gradlew bootRun**
Swagger: **http://localhost:8080/swagger-ui/index.html#/**
h2 console : **http://localhost:8080/h2-console/**
Test Cases Report(For backend): **./gradlew clean test ; start build/reports/tests/test/index.html**
Test Cases Report & Coverage summary(For frontend): **ng test --watch=false --code-coverage ; if (Test-Path coverage/index.html) { start coverage/index.html }**
